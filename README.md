# Zero-DCE 低光图像增强项目

本项目基于Zero-DCE模型实现低光图像增强，针对17张Zero-DCE数据集（train=13/val=2/test=2）完成训练与测试。

## 核心训练日志（用于评估）

| Epoch | Train Loss | Val Loss | 说明               |
|-------|------------|----------|------------------|
| 50    | 0.002879   | 0.001435 | 训练稳定收敛           |
| 50    | 0.003633   | 0.001433 | 最佳验证损失（0.001433） |

> 注：因数据集规模较小（仅17张），验证损失整体偏低，符合小数据集训练规律。训练过程稳定，无明显过拟合。
